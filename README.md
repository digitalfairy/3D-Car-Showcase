An immersive, high-fidelity 3D car exhibition platform. This project demonstrates advanced capabilities in **Three.js** and **React Three Fiber**, focusing on realistic vehicle rendering, environment orchestration, and seamless camera transitions between multiple 3D scenes.

---

## ‚ú® Project Showcase & Technical Focus

This application serves as a technical demonstration of modern 3D web development:

* **High-Fidelity Vehicle Rendering:** Optimized loading and material application for complex GLB/GLTF car models (Tesla Model 3, Semi, and Cybertruck).
* **Cinematic Transitions:** Implementation of smooth, state-driven camera movements that transition the user through distinct environmental "stages."
* **Advanced Post-Processing:** Utilizes `@react-three/postprocessing` to implement **Bloom**, **SSR (Screen Space Reflections)**, and **Depth of Field** for a realistic automotive showroom feel.
* **Environment Logic:** A modular system for swapping HDRIs, lighting setups, and ground reflections dynamically using **Jotai** state.
